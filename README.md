# Running Dashboard

## Project Goal

To analyse personal running data exported from Apple Health and create an interactive dashboard showing training trends and performance metrics.

## Metrics

- Total Runs
- Total Distance
- Longest Run
- Average Pace

## Visualisations

- Distance by Month
- Runs by Weekday
- Distance Over Time

## Future Development

- Weight vs Pace analysis
- Training Volume vs Pace
- Half Marathon preparation tracking
